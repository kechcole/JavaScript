1. Changing the DOM.

1.1 Creating New Nodes

There are various ways in which the DOM can be changed. Static websites require one to directly change elements in the HTML file but dynamic websites are use JS.
A combination of createElement() method and textContent property will give you a textual content. But these new features are not visible in the front end of the web browser until when they are inserted. With a combination of appendChild(), insertBefore(), and replaceChild(), you can insert nodes and elements anywhere in the DOM.

1.2 Inserting Nodes into the DOM

We need to insert created elements and nodes into the document for it to be seen by either adding or replacing.

1.3 Removing Nodes from the DOM.

Now we know how to create elements, add them to the DOM, and modify existing elements. The final step is to learn to remove existing nodes from the DOM. Child nodes can be removed from a parent with removeChild(), and a node itself can be removed with remove().
