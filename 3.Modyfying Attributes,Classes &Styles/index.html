1.1 Modify Attributes.


Attributes provide additional information about an element, though not all do contaion them. The must be specified the starting tag using name and value pairs i.e class='id' .
In JS we have multiple methods to modify them :

  • hasAttribute() - Returns a boolean value
  • getAttribute() - Return a value of a specified attribute
  • setAttribute() - Adds / updates a value of a specified attribute.
  • removeAttribute() - Remove an attribute from the element.
